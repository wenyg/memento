# 版本历史

## 0.2.0（待发布）

- ✨ **新增 TODO 管理功能**
  - 支持解析 Markdown 任务列表（`- [ ]` 和 `- [x]`）
  - 支持嵌套层级的 TODO 项
  - 扩展属性支持：优先级（priority:H/M/L）、项目（project:xxx）、截止日期（due:YYYY-MM-DD）、标签（#tag）
  - 多种分组方式：按文件、按项目、按优先级、按状态
  - 一键跳转到 TODO 所在文件的具体行
  - 快速切换 TODO 完成状态
  - TODO 视图按钮和工具栏操作
- ✅ 添加 TODO 功能文档和使用说明
